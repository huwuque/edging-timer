# ⏱ 寸止计时器

一个基于 Python tkinter 的桌面计时工具，专为记录"寸止"（Edging）训练设计。

## ✨ 功能

- **主计时器** — 记录整个 session 的总时长
- **寸止计时** — 快要高潮时按下"寸止"按钮，开始独立计时；忍耐期结束后再按一次，记录本次寸止时长
- **多次寸止记录** — 每次寸止的时长都会被记录下来，支持历史查看
- **暂停 / 继续** — 随时暂停和恢复主计时
- **重置** — 一键清空所有数据回到初始状态

## 🚀 运行方式

### 环境要求

- Python 3.6+
- 无需安装任何第三方库（仅使用 Python 标准库 tkinter）

### 启动

```bash
python edging_timer.py
```

或直接双击 `edging_timer.py` 文件。

## 🎮 使用说明

| 按钮 | 作用 |
|------|------|
| **开始** | 启动主计时器 |
| **暂停 / 继续** | 暂停或恢复主计时 |
| **寸止** | 进入寸止模式（按钮变红）；再次点击退出并记录本次时长 |
| **重置** | 清空所有计时和记录，回到初始状态 |

## 🧠 状态机

```
IDLE ──(开始)──> RUNNING ──(寸止)──> EDGING ──(停止寸止)──> RUNNING
  ^                  │                                         │
  └──(重置)──────────┴──(重置)─────────────────────────────────┘
                     RUNNING ──(暂停)──> PAUSED ──(继续)──> RUNNING
```
