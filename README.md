# ⏱ 寸止计时器

一个基于 Python tkinter 的桌面计时工具，专为记录"寸止"（Edging）训练设计。

## ✨ 功能

- **主计时器** — 记录整个 session 的总时长
- **奖励图片** — 主计时每过 10 秒自动弹出奖励图片，2 秒后消失
- **寸止计时** — 快要高潮时按下"寸止"按钮，开始独立计时；忍耐期结束后再按一次，记录本次寸止时长
- **多次寸止记录** — 每次寸止的时长都会被记录下来，支持历史查看
- **暂停 / 继续** — 随时暂停和恢复主计时
- **重置** — 一键清空所有数据回到初始状态
- **高清渲染** — 启用 Windows Per Monitor DPI v2，文字清晰不模糊
- **窗口自由缩放** — 可随意拖拽调整窗口大小

## 🎨 主题

莫兰迪配色（Morandi Palette）— 低饱和度、灰调、温暖柔和

| 元素 | 颜色 |
|------|------|
| 背景 | 暖灰白 `#EDEAE5` |
| 主按钮 | taupe 暖灰褐 `#B0A8A0` |
| 寸止中 | 尘玫瑰 `#C4A5A5` |
| 文字 | 暖深灰褐 `#4E4A46` |

## 🚀 运行方式

### 环境要求

- Python 3.6+
- Pillow（用于加载奖励图片）

```bash
pip install Pillow
```

### 启动

```bash
python edging_timer.py
```

或直接双击 `edging_timer.py` 文件。

## 🎮 使用说明

| 按钮 | 作用 |
|------|------|
| **开始** | 启动主计时器，每 10 秒弹出奖励图 |
| **暂停 / 继续** | 暂停或恢复主计时 |
| **寸止** | 进入寸止模式（按钮变尘玫瑰色）；再次点击退出并记录本次时长 |
| **重置** | 清空所有计时、记录和奖励里程碑 |

## 🖼️ 奖励图片

将奖励图片命名为 `奖励图片.jpg` 放在项目目录下即可。主计时每过 10 秒（10s、20s、30s…）自动居中弹出，2 秒后消失。暂停期间不触发。

## 🧠 状态机

```
IDLE ──(开始)──> RUNNING ──(寸止)──> EDGING ──(停止寸止)──> RUNNING
  ^                  │                                         │
  └──(重置)──────────┴──(重置)─────────────────────────────────┘
                     RUNNING ──(暂停)──> PAUSED ──(继续)──> RUNNING
```

## 🛠️ 技术栈

- **语言**: Python 3
- **GUI**: tkinter（Python 标准库）
- **图片处理**: Pillow
- **高 DPI**: ctypes + SetProcessDpiAwareness(2)
- **主题**: 莫兰迪浅色高级配色
